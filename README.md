# C++AntiVirus

This program I am making in Visual Studio Code
<br>
## Build project
1. Clone this project
   ```
    git clone https://github.com/kecalek13/CppAntiVirus
   ```
2. Go to CppAntiVirus
   ```
    cd CppAntiVirus
   ```
4. Build it using g++
   ```
    g++ main.cpp -o main.exe
   ```
